# V1-3-21 check-invoice-special-rules (インボイス特例群 skill)

## 1. skill 概要

インボイス制度において「**帳簿保存のみで仕入税額控除を許容する例外群**」を束ねる skill。
税理士が顧問先データをレビューする際に、**過剰に適格請求書を求めてしまう状況を抑止する** ことが本筋。

既存 V1-3-10/11 (false negative world = 見落としを減らす) と対をなし、本 skill は **false positive world**
(過剰確認を減らす) に属する。

## 2. 対象特例 (4 系統)

| 系統 | 対象 | 金額閾値 | 期間制限 |
|---|---|---|---|
| IS-01 少額特例 | 1万円未満 (税込) の課税仕入 | 1 万円未満 | R5/10/1 - R11/9/30 |
| IS-02 公共交通機関特例 | 鉄道・路線バス・船舶 | 3 万円未満 | 期限なし |
| IS-03 自販機特例 | 自販機・自動サービス機 | 3 万円未満 | 期限なし |
| IS-04 出張旅費特例 | 従業員への出張旅費精算 | 通常必要分 | 期限なし |

## 3. 検出 ID 一覧

| ID | 内容 | severity | confidence | error_type |
|---|---|---|---|---|
| IS-01a | 少額特例適用可能性 (advisory) | 🟢 Low | 50 | gray_review |
| IS-01b | 少額特例期間外発動 (R11/10 以降) | 🟡 Medium | 90 | mild_warning |
| IS-02a | 公共交通機関特例適用可能性 | 🟡 Medium | 70 | gray_review |
| IS-02b | 公共交通機関特例対象外 (タクシー等) | 🟠 High | 80 | invoice_warning |
| IS-03a | 自販機特例適用可能性 | 🟢 Low | 60 | gray_review |
| IS-04a | 出張旅費特例適用可能性 | 🟡 Medium | 60 | gray_review |

(IS-04b 通常必要範囲超過は 039 スコープ外、第 10 ラウンド以降検討)

## 4. advisory 性質 (false positive world として)

特に IS-01a / IS-03a は **advisory finding** (severity=🟢 Low + confidence 低め) で実装。
「適用可能性がある」を税理士に提示するが、確認を強要しない設計。

理由:
- 件数が多発しやすい (IS-01a は 1万円未満すべてが対象、推定 200-500 件/6ヶ月)
- 顧問先疲弊抑制を最優先
- 業務確認は税理士判断で「全件確認」「サンプル確認」「無視」を選択可能

これは V1-3-10/11 の direct_error (🔴 Critical) とは性質が異なる。

└─ かっこ書き「請求書等の交付を受けることが困難な場合 (中略) 政令で定める場合」
   │
   ├─ 消費税法施行令 第49条第1項第1号
   │  ├─ イ (公共交通機関特例の買手側保存特例)
   │  │   └─ 施行令第70条の9第2項第1号 → IS-02 公共交通機関特例
   │  │       (3万円未満、鉄道/路線バス/船舶/軌道、★タクシー除外)
   │  │
   │  └─ ニ (財務省令で定める課税仕入れ)
   │       └─ 施行規則 第15条の4
   │           ├─ 第1号 → 施行規則 第26条の6第1号 → IS-03 自販機特例
   │           │   (3万円未満、自動販売機・自動サービス機)
   │           └─ 第2号 → IS-04 出張旅費特例
   │               (通常必要分、従業員への支給金品)

   └─ 平成28年改正法附則 第53条の2 → IS-01 少額特例 (経過措置)

# 消費税区分マスタ

**目的**: freee会計で使用される消費税区分の一覧と判定基準を定義する。
**更新頻度**: 税制改正時（年1回程度）
**参照元Skill**: `check-tax-classification`, `check-reduced-tax-rate`, `check-overseas-service-tax` 他V1-3系

---

## 1. 消費税区分の基本分類

### 1.1 売上側（課税売上）

| freee税区分コード | 名称           | 税率 | 備考                                           |
| ----------------- | -------------- | ---- | ---------------------------------------------- |
| 課税売上10%       | 課対売上10%    | 10%  | 標準税率                                       |
| 課税売上8%(軽減)  | 課対売上8%(軽) | 8%   | 食品・新聞（週2回以上発行）                    |
| 非課税売上        | 非課売上       | -    | 住宅家賃、土地の譲渡、有価証券など             |
| 輸出売上          | 輸出売上       | 0%   | 輸出免税（書類要件あり）                       |
| 対象外            | 不課税売上     | -    | 対価性のない収入（受取配当金、受取保険金など） |

### 1.2 仕入側（課税仕入）

| freee税区分コード | 名称               | 税率 | 備考                           |
| ----------------- | ------------------ | ---- | ------------------------------ |
| 課税仕入10%       | 課対仕入10%        | 10%  | 標準税率、適格請求書必要       |
| 課税仕入8%(軽減)  | 課対仕入8%(軽)     | 8%   | 食品・新聞                     |
| 非課税仕入        | 非課仕入           | -    | 住宅家賃、土地、行政手数料など |
| 対象外            | 不課税仕入         | -    | 給与、香典、見舞金など         |
| 課対輸本          | 課対仕入(輸入本体) | 10%  | 輸入商品本体                   |
| 地消貨割          | 地方消費税貨物割   | -    | 輸入時の地方消費税             |

## 3. 判定の優先順位

同じ科目でも税区分が分かれるケースでは、以下の順で判定する：

1. **取引先の国内/国外** → 国外事業者への支払いは海外サービス消費税チェック（V1-3-40）へ
2. **物品 or サービス** → 物品の輸入は課対輸本、国外からのサービスはリバースチャージ
3. **軽減税率対象かどうか** → 食品・定期購読新聞は8%
4. **非課税取引に該当するか** → 住宅家賃、土地、金融取引、医療、教育など
5. **対価性があるか** → 無ければ不課税（給与、香典、補助金など）

## 4. よくある誤り

| 誤りパターン                        | 正しい処理                                   |
| ----------------------------------- | -------------------------------------------- |
| 住宅家賃を課対仕入10%で計上         | 非課仕入                                     |
| 収入印紙を課対仕入10%で計上         | 非課仕入                                     |
| 海外のSaaS利用料を課対仕入10%で計上 | リバースチャージ（特定課税仕入）             |
| 新聞代をすべて8%で計上              | 週2回以上発行の定期購読のみ8%、それ以外は10% |
| 軽油購入を全額課対仕入10%で計上     | 軽油引取税部分は対象外                       |
| 海外出張の航空券を課対仕入10%で計上 | 国際線部分は輸出免税（仕入側は対象外）       |

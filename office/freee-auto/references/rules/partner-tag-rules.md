# 取引先タグルール

自動登録ルールCSVの「取引先」（カラム12）の生成ルール。

---

## 1. 取引先タグ必須の勘定科目

以下の科目では必ず取引先タグを付ける:

- 売上高
- 仕入高
- 外注費 / 士業報酬
- 賃借料 / 地代家賃
- 役員貸付金 / 役員借入金

## 2. 取引先タグ推奨の勘定科目

- 通信費等の毎月発生する費用 → 計上がない月に気づきやすくするため
- インボイス登録業者 → すべての支払い取引に取引先タグ必須

## 3. 取引先名の命名規則

### 法人
- 略称ルール適用（input-general-rules.md参照）
- 例: ㈱アックスコンサルティング

### 個人事業主（屋号あり）
- 「個人名(屋号)」形式
- 例: 山田花子(山田デザイン)

### 飲食店
- 「店名(法人名)」または「店名(個人名)」
- 不明な場合は店名のみ
- 例: アンソラス(㈱オールドトライアングル)

### 地代家賃・賃借料
- 「大家名(大家名 or 管理会社)」形式
- 貸主を取引先にする（管理会社ではなく）
- 例: 明日太郎(上野不動産㈱)

### インボイス未登録事業者
- 「会社名(未登録)」形式
- 例: 国税商事㈱(未登録)

### タクシー（アプリ利用）
- アプリ名を取引先にする
- 例: GO、S.RIDE、Uber

### 不特定多数の個人売上（小売等）
- 「個人顧客」を取引先にする

### 個人名
- スペースなし: 明日太郎

## 4. ショートカット設定（freee取引先マスタ）

新規取引先登録時に必ず設定:
- ショートカット1: ローマ字（例: kokuzei）
- ショートカット2: ひらがな（例: こくぜい）

※自動登録ルールCSVでは取引先名のみセット。
  ショートカットは現状スタッフがfreee取引先マスタで設定。
  将来的にはClaude Code + freee APIで自動設定を目指す。

## 5. 重複取引先の管理

- 削除できる場合: 削除
- 削除できない場合: 正しい方を残し、もう一方を「【未使用】○○」として使用停止/アーカイブ

## 6. インボイス登録確認

新規取引先は必ずfreee会計の「取引先設定」で適格請求書発行事業者かどうかを確認し、登録番号を登録する。

## 7. 相手先が特定できない場合

- ららぽーと、東急、松坂屋など商業施設名のみ → 取引先タグをつけない
- 消費税は「対象外仕入」とする

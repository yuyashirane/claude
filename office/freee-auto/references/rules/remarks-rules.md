# 備考欄ルール

自動登録ルールCSVの「備考」（カラム22）の生成ルール。

---

## 1. 基本方針

- 預金明細・クレカ明細 → 備考欄を入力する
- 立替経費・現金出納帳 → 備考欄を入力しない
- 自動登録ルールでは定型的な内容のみセット（「○月分」等の動的情報は省略）

## 2. 備考欄記載の閾値

### 一般経費: 税込5万円以上

| 勘定科目 | 備考欄の記載内容 |
|---------|----------------|
| 交際費・会議費 | 品目タグありなら不要。なければ相手先・人数・目的 |
| 消耗品費・固定資産 | 内容と数量（例: 椅子3脚） |
| 旅費交通費（飛行機） | 区間（例: 羽田-ロンドン） |
| 旅費交通費（ホテル） | 海外の場合のみ国名 |
| 生命保険料 | 経理処理（全額損金、1/2損金等） |

### 外注費: 税込30万円以上

- 内容を備考欄に記載

## 3. 自動登録ルールCSVでの備考欄

自動登録ルールで備考欄に自動セットする内容は限定的:
- 金額条件に基づく動的な内容は自動セット不可（人が個別判断）
- 取引先名や品目で十分な情報がある場合は備考欄は空でよい

## 4. 少額特例（税込1万円未満）

基準期間の課税売上高が1億円以下の事業者の場合:
- 税込1万円未満で詳細が分からない → 取引先・品目・備考すべてブランク
- 消費税は通常の課税仕入で処理

## 5. 不明取引の処理フロー

### 相手先が特定できない場合
- 商業施設名（ららぽーと、東急、松坂屋等）→ 取引先タグなし
- 消費税は「対象外仕入」

### 勘定科目が不明な場合
- 「未確定(損益)」を使用
- 仮払金・仮受金は使用しない

# 品目タグルール

自動登録ルールCSVの「品目」（カラム16）の生成ルール。

---

## 1. 標準品目タグ一覧

以下は全顧客共通の標準品目タグ。freee会計利用開始時に必ずアップロードする。

### 給与・社会保険系（自動登録ルール対象外）
- 基本給、残業代、通勤手当
- 健康保険料、厚生年金保険料、雇用保険料、介護保険料、住民税
- 源泉所得税（給与）、源泉所得税（利息）、源泉所得税（士業）
- 源泉所得税（賞与）、源泉所得税（配当）、源泉所得税（外注・士業以外）、源泉所得税（売上）

### 交通費系
- 電車代、バス代、タクシー代、高速代、飛行機代
- ガソリン代、ガソリン代（共通）

### 光熱費・通信系
- 電気代、電気代（共通）
- ガス代、ガス代（共通）
- 水道代、水道代（共通）
- 電話代、電話代（共通）
- 携帯代、携帯代（共通）
- インターネット代、インターネット代（共通）

### 事務所・住居系
- 事務所賃料、事務所賃料（共通）
- 自宅兼事務所、社宅賃料
- 駐車場代、駐車場代（共通）、駐車場賃料

### 事務・IT系
- 事務用品、事務用品（共通）
- ソフト代（パッケージソフト・ライセンス）
- ツール代（SaaS・クラウドサービス）

### 税金系（自動登録ルール対象外）
- 法人税、地方税、事業税、固定資産税、自動車税、消費税

### その他
- 飲食代、書籍代、宿泊代、贈答品、収入印紙
- 郵便代、配送料、税理士報酬、振込手数料、慶弔費

## 2. 品目タグ必須の勘定科目

| 勘定科目 | 品目タグ例 |
|---------|-----------|
| 預り金 | 源泉所得税（給与/利息/士業/賞与/配当/外注・士業以外/売上） |
| 短期借入金/長期借入金/リース債務/長期未払金/支払利息 | 金融機関名+丸数字_元本_返済額（例: 公庫①_10M_166K） |
| 保険積立金 | 保険会社名+丸数字_被保険者名 |

## 3. 品目タグ推奨の勘定科目

- 支払手数料、旅費交通費、会議費、交際費

## 4. 特殊な品目タグ命名規則

| 対象 | 命名規則 | 例 |
|------|---------|-----|
| 借入金・支払利息 | 金融機関名+丸数字_借入時の元本_毎月の返済額 | 公庫①_10M_166K |
| 保険積立金 | 保険会社名+丸数字_被保険者名 | 東京海上①_明日太郎 |
| 交通反則金・延滞税・加算税 | 「罰金等」固定 | 罰金等 |
| 更新料・礼金 | 「更新料・礼金」固定 + 備考欄に詳細 | 更新料・礼金 |
| 法人税等の支払 | 法人税/事業税/住民税を個別に | 法人税 |
| 雑収入（件数多い場合） | 利子補給、○○助成金、○○補助金等 | 雇用調整助成金 |

## 5. （共通）バリエーションの使い分け

- **法人顧客**: 「（共通）」なしの品目を使用（例: 電気代）
- **個人事業主**: 「（共通）」ありの品目を使用（例: 電気代（共通））
  - 理由: freeeの「家事按分」処理で事業用/私用を按分するため

## 6. 自動登録ルールCSVでの品目タグ対象

自動登録ルールで品目タグを自動セットする対象:
- **対象**: 経費系の品目（交通費、光熱費、通信費、事務用品、IT系、その他）
- **対象外**: 給与・社会保険系、税金系（これらは給与仕訳・決算仕訳で処理）

# 信頼度スコア算出ルール

仕訳候補の信頼性を5要素・100点満点で評価する。

---

## スコア要素と配点

### 要素1: キーワード辞書マッチ（30点満点）

`references/accounting/account-dictionary.md` との照合結果。

| 条件                                                  | 得点 |
| ----------------------------------------------------- | ---- |
| 完全一致（科目名が摘要に含まれる）                    | 30pt |
| キーワード一致（辞書のキーワードにマッチ、候補が1つ） | 25pt |
| キーワード一致（候補が2つ）                           | 15pt |
| キーワード一致（候補が3つ以上）                       | 8pt  |
| 不一致                                                | 0pt  |

### 要素2: 過去仕訳パターン一致（30点満点）

freeeの過去仕訳データとの照合結果。

| 条件                                              | 得点 |
| ------------------------------------------------- | ---- |
| 同一取引先 ＋ 類似摘要 ＋ 同一科目で3件以上の実績 | 30pt |
| 同一取引先 ＋ 類似摘要で1〜2件の実績              | 20pt |
| 同一取引先のみ一致（摘要は異なる）                | 15pt |
| 類似摘要のみ一致（取引先は異なる）                | 8pt  |
| 過去実績なし                                      | 0pt  |

「類似摘要」の判定: 摘要の主要キーワード（名詞・固有名詞）が70%以上一致。

### 要素3: 金額の妥当性（15点満点）

推定した勘定科目の通常金額レンジとの比較。

| 条件                                              | 得点          |
| ------------------------------------------------- | ------------- |
| レンジ内（過去の同一科目の平均±2σ以内）           | 15pt          |
| 境界（平均±2σ〜3σ）                               | 10pt          |
| レンジ外（平均±3σ超、または科目別の常識的範囲外） | 5pt           |
| 金額データ不足で判定不能                          | 8pt（中間値） |

科目別の常識的範囲の例:

- 旅費交通費: 100円〜50万円
- 通信費: 100円〜30万円
- 消耗品費: 100円〜10万円未満
- 水道光熱費: 1,000円〜100万円

### 要素4: 消費税ルール明確さ（15点満点）

R01〜R12ルールでの判定結果。

| 条件                                            | 得点 |
| ----------------------------------------------- | ---- |
| 一意に判定可能（1つのルールで確定）             | 15pt |
| 概ね判定可能（デフォルト課税10%で問題なさそう） | 12pt |
| 複数候補あり（課税/非課税の判断が微妙）         | 8pt  |
| 判定困難（摘要の情報不足、特殊取引）            | 3pt  |
| 判定不能                                        | 0pt  |

### 要素5: 摘要の情報量（10点満点）

摘要テキストに含まれる情報の充実度。

| 条件                                                   | 得点 |
| ------------------------------------------------------ | ---- |
| 取引先名 ＋ 取引内容 ＋ 数量/期間                      | 10pt |
| 上記のうち2要素                                        | 7pt  |
| 上記のうち1要素                                        | 3pt  |
| 摘要なし、または「カード払い」等の無意味なテキストのみ | 0pt  |

## 判定閾値

| ランク     | スコア範囲 | 処理                                                                                 |
| ---------- | ---------- | ------------------------------------------------------------------------------------ |
| **High**   | 75〜100点  | 自動登録候補。Kintoneに「未レビュー」で登録。若手が一括確認→承認。                   |
| **Medium** | 45〜74点   | スタッフレビュー対象。Kintoneのスタッフレビュー一覧に表示。個別に確認して承認/修正。 |
| **Low**    | 0〜44点    | シニアレビュー対象。Kintoneのシニアレビュー一覧に表示。判断＋修正。                  |

---

## 特殊フラグ

スコアとは別に、以下のフラグを付与する場合がある:

- **「判定不能」**: いずれのキーワードにもマッチせず、過去実績もない → スコア0点
- **「消費税要確認」**: R01〜R12のいずれかに該当する指摘がある → 該当RxxのIDを記録
- **「固定資産確認」**: 10万円以上の消耗品費、20万円以上の修繕費 → 資本的支出の検討
- **「源泉徴収確認」**: 士業・デザイン・原稿料等の外注費 → 源泉徴収の要否確認
- **「インボイス要確認」**: インボイス区分が「要確認」の取引

## フィードバックループ

人間がKintoneで修正した内容を、スコアリングの改善に活用する:

1. Kintoneの「修正科目」「修正税区分」「修正理由」を収集
2. 修正パターンを分析（どの要素のスコアが実態とズレていたか）
3. キーワード辞書の追加・修正を提案
4. 月次で精度レポートを生成
   - High ランクの正答率（目標: 95%以上）
   - Medium ランクの修正率
   - Low ランクの判定不能率
5. 目標: 半年後にHigh率80%以上

### 実装上のスコア算出ロジック

| 要素 | 実装の判定方法 |
|------|---------------|
| キーワード辞書マッチ(30pt) | 一意マッチ+最長KW≥4文字=30pt, ≥2文字=25pt, <2文字=20pt, 2科目=15pt, 3科目以上=8pt |
| 過去仕訳パターン(30pt) | 常時0pt（未実装） |
| 金額妥当性(15pt) | <10万=15pt, 10-30万=10pt, ≥30万=5pt |
| 消費税ルール明確さ(15pt) | R01-R12判定結果による（売上高=15pt確定、海外サービス=10-12pt等） |
| 摘要情報量(10pt) | ≥20文字=10pt, 10-19文字=7pt, 5-9文字=4pt, <5文字=0pt |

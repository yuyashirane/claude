# Notionレビューダッシュボード仕様

仕訳候補のレビュー・承認・修正を管理するNotionデータベースの仕様。

---

## データベース概要

- DB ID: （実際のIDに置き換える）
- 用途: Claude Codeが生成した仕訳候補を人間がレビュー・承認するためのダッシュボード
- プロパティ数: 20

---

## プロパティ一覧

| # | プロパティ名 | 型 | 説明 |
|---|-------------|------|------|
| 1 | 取引日 | Date | 取引の発生日 |
| 2 | 金額 | Number | 取引金額（税込） |
| 3 | 摘要 | Rich text | freeeの摘要テキスト |
| 4 | 推定勘定科目 | Select | AIが推定した勘定科目（22種） |
| 5 | 推定消費税区分 | Select | AIが推定した消費税区分（9種） |
| 6 | 信頼度スコア | Number | 0〜100の信頼度点数 |
| 7 | 信頼度ランク | Select | High / Medium / Low |
| 8 | インボイス区分 | Select | 適格 / 非適格80% / 非適格50% / 不要 / 要確認 |
| 9 | 消費税指摘 | Multi-select | R01〜R12（該当するルールID） |
| 10 | レビューステータス | Select | 未レビュー / レビュー中 / 承認済み / 修正済み / 却下 |
| 11 | レビュー担当 | Person | レビュー担当者 |
| 12 | 修正科目 | Select | 人間が修正した勘定科目（修正がある場合のみ） |

| 14 | 修正理由 | Rich text | 修正した理由（フィードバックループ用） |

| 16 | freee明細ID | Rich text | freee上の明細ID（紐付け用） |

| 20 | 備考 | Rich text | その他のメモ |

## 勘定科目の選択肢（22種）

売上高、仕入高、外注費、給料手当、役員報酬、法定福利費、福利厚生費、地代家賃、租税公課、支払利息、保険料、交際費、会議費、消耗品費、支払手数料、旅費交通費、通信費、水道光熱費、新聞図書費、広告宣伝費、修繕費、雑費

## 消費税区分の選択肢（9種）

課税10%、課税8%（軽減）、非課税、不課税、免税、対象外、課税10%（インボイス経過80%）、課税10%（インボイス経過50%）、要確認

## ビュー構成（3ビュー）

### 🌱 若手レビュービュー
- **表示形式**: テーブル
- **フィルター**: 信頼度ランク = High or Medium、レビューステータス = 未レビュー
- **ソート**: 信頼度スコア 降順
- **用途**: Highはバッチ承認（一括で「承認済み」に変更）、Mediumは個別確認
- **表示列**: 取引日、金額、摘要、推定勘定科目、推定消費税区分、信頼度スコア、インボイス区分

### 💼 経験者レビュービュー
- **表示形式**: カンバン（信頼度ランク別）
- **フィルター**: 信頼度ランク = Low、または消費税指摘が空でない、レビューステータス = 未レビュー
- **ソート**: 消費税指摘のあるものを優先
- **用途**: 判断が必要な案件に集中。修正科目・修正税区分・修正理由を記入。
- **表示列**: 取引日、金額、摘要、推定勘定科目、推定消費税区分、信頼度スコア、消費税指摘、インボイス区分

### 📊 進捗管理ビュー
- **表示形式**: テーブル（顧問先別にグループ化）
- **フィルター**: なし（全件表示）
- **ソート**: 顧問先名 → レビューステータス
- **用途**: 顧問先ごとの処理状況を俯瞰。AI精度の把握。
- **表示列**: 顧問先名、レビューステータス（集計）、修正率（修正済み÷全件）

---

## ステータス遷移

```
未レビュー → レビュー中 → 承認済み → （freeeに仕訳登録）
                        → 修正済み → （freeeに修正仕訳登録）
                        → 却下    → （処理しない）
```

---

## フィードバックデータの活用

月次で以下のレポートを生成:
- **AI正答率**: 承認済み（修正なし）÷ 全件
- **修正率**: 修正済み ÷ 全件
- **よくある修正パターン**: 修正理由の集計
- **科目別精度**: 勘定科目ごとの正答率
- **消費税指摘の的中率**: 指摘があった取引のうち実際に修正が必要だった割合

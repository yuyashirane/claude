# freee 自動登録ルールCSV仕様書

## 基本情報

| 項目 | 値 |
|------|-----|
| エンコーディング | Shift_JIS (cp932) |
| 改行 | CRLF |
| カラム数 | 53列 |
| 1行目 | ヘッダー行 |
| 重複判定キー | カラム[0]〜[6]の7項目が完全一致 → 上書き、不一致 → 新規登録 |

---

## カラム定義（全53列）

### 条件側（7列 — 重複判定キー）

| # | カラム名 | 説明 |
|---|---------|------|
| [0] | 収支区分 | 「収入」or「支出」【必須】 |
| [1] | 取引口座 | freee口座名。空=全口座対象 |
| [2] | カードラベル | クレカのカード名。通常空 |
| [3] | 取引内容 | 明細の摘要テキスト【必須】※原文をそのまま（全角含む） |
| [4] | マッチ条件 | 「完全一致」「部分一致」「前方一致」【必須】 |
| [5] | 金額（最小値） | 金額範囲下限。通常空 |
| [6] | 金額（最大値） | 金額範囲上限。通常空 |

### 処理側（主要カラム）

| # | カラム名 | 説明 |
|---|---------|------|
| [7] | 優先度 | 数値。通常「0」 |
| [8] | マッチ後のアクション | 【必須】下記「マッチ後のアクション一覧」参照 |
| [9] | 振替口座 | 口座振替時のみ使用 |
| [10] | 取引テンプレート | テンプレート名。通常空 |
| [11] | 購入データ原本に準拠 | 通常空 |
| [12] | 取引先 | freee取引先名 |
| [13] | 適格請求書等 | 「取引先情報に準拠」「該当する」「該当しない」 |
| [14] | 勘定科目 | freee上の科目名（例: 通信費、支払手数料） |
| [15] | 税区分 | freee上の税区分文字列（下記参照） |
| [16] | 品目 | freee品目タグ |
| [17] | 部門 | freee部門タグ |
| [18] | メモタグ（複数指定可・カンマ区切り） | |
| [19] | セグメント1 | |
| [20] | セグメント2 | |
| [21] | セグメント3 | |
| [22] | 備考 | |

### 消込関連（カラム[23]〜[52] 計30列）

| # | カラム名 | 説明 |
|---|---------|------|
| [23] | 消込を実行する際の差額 | |
| [24]〜[33] | 過小入金・出金関連 | 10列 |
| [34]〜[42] | 過小入金・出金の複合差額関連 | 9列 |
| [43]〜[52] | 過剰入金・出金関連 | 10列 |

> 通常の取引登録/推測ルールでは全て空。Claude Codeでは生成しない。

---

## マッチ後のアクション一覧

| 値 | 用途 | Claude Code生成対象 |
|----|------|-------------------|
| 取引を登録する | 完全自動登録（消込自動完了） | ○ 高信頼＋安全 |
| 取引を推測する | 科目プリセット→人がワンクリック確認 | ○ 中信頼度 |
| 振替を推測する | 口座振替の推測 | △ 将来対応 |
| 振替を登録する | 口座振替の自動登録 | △ 将来対応 |
| 消込を推測する | 未決済取引消込の推測 | × |
| 取引テンプレートを推測する | 複合仕訳推測 | × |
| 無視する取引を推測する | 無視する明細の推測 | × |

## 税区分（パイプライン出力 → freee CSV値）

| パイプライン出力 | freee CSV値 | 備考 |
|----------------|------------|------|
| 課対仕入10% | 課対仕入 | freeeが取引日から税率自動判定 |
| 課対仕入8%（軽） | 課対仕入8%（軽） | |
| 非課仕入 | 非課仕入 | |
| 非課税仕入 | 非課仕入 | |
| 対象外 | 対象外 | |
| 不課税 | 不課税 | |
| 課税売上10% | 課税売上 | freeeが取引日から税率自動判定 |
| 非課売上 | 非課売上 | |

> **重要:** 「課対仕入」と書くだけでfreeeが自動判定する。10%を明記する必要はない。

---

## 適格請求書等（パイプライン出力 → freee CSV値）

| パイプライン出力 | freee CSV値 | 備考 |
|----------------|------------|------|
| 適格 | 取引先情報に準拠 | 取引先マスタの設定に従う |
| 非適格80% | 該当しない | |
| 非適格50% | 該当しない | |
| 不要 | 該当しない | |
| 要確認 | （空） | 人が判断 |

> **重要:** 「取引先情報に準拠」が最も安全。取引先名さえ正しければインボイス判定はfreeeに委ねられる。

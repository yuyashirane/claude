# インボイス判定ルール

課税仕入の取引に対するインボイス区分の判定ルール。
CLAUDE.md の「インボイス制度対応」セクションの詳細版。

---

## 判定フロー

```
課税仕入か？
├── No → 区分「不要」（不課税・非課税は対象外）
└── Yes
    ├── 国外取引か？ → 区分「不要」
    ├── 3万円未満の公共交通機関か？ → 区分「不要」
    ├── 自動販売機（3万円未満）か？ → 区分「不要」
    ├── 郵便切手か？ → 区分「不要」
    └── 取引先マスタにインボイス登録番号があるか？
        ├── Yes → 区分「適格」（全額控除）
        └── No
            ├── 取引日が2026/9/30以前か？ → 区分「非適格（80%控除）」
            ├── 取引日が2029/9/30以前か？ → 区分「非適格（50%控除）」
            ├── それ以降 → 区分「非適格（控除不可）」
            └── 判定不能 → 区分「要確認」
```

---

## インボイス区分の定義

### 適格
- **条件**: 取引先がインボイス発行事業者（登録番号 T + 13桁）
- **控除率**: 100%
- **必要書類**: 適格請求書（インボイス）の保存

### 非適格（経過措置80%）
- **適用期間**: 2023/10/1 〜 2026/9/30
- **控除率**: 80%
- **必要書類**: 区分記載請求書の保存 + 帳簿に「80%控除対象」の記載

### 非適格（経過措置50%）
- **適用期間**: 2026/10/1 〜 2029/9/30
- **控除率**: 50%
- **必要書類**: 区分記載請求書の保存 + 帳簿に「50%控除対象」の記載

### 不要
以下の取引はインボイスの保存が不要（帳簿のみで仕入税額控除可能）:
- 3万円未満の公共交通機関（鉄道、バス、船舶）
- 3万円未満の自動販売機・自動サービス機
- 郵便切手を対価とする郵便サービス
- 従業員等に支給する通常必要な出張旅費・宿泊費・日当
- 国外取引
- 古物営業法に基づく古物の購入（一定要件あり）

### 要確認
- 上記いずれにも該当しない場合
- 取引先のインボイス登録状況が不明な場合
- → 人間レビューへエスカレーション

## 少額特例（1万円未満）

**適用要件**:
- 基準期間の課税売上高が1億円以下
- または、特定期間の課税売上高が5,000万円以下
- 税込1万円未満の課税仕入（1回の取引の合計額）

**適用期間**: 2023/10/1 〜 2029/9/30

**効果**: インボイスの保存なしで仕入税額控除が可能（帳簿のみでOK）

## freeeでの処理

freeeの取引登録時に以下を設定:
- **税区分**: 通常の課対仕入10%等を選択
- **インボイス**: freeeの「インボイス」欄で区分を選択
  - 適格 → 「適格請求書」
  - 80%控除 → 「80%控除」
  - 50%控除 → 「50%控除」
  - 不要 → 「帳簿のみ保存」
  - 要確認 → 空欄のまま（後で人間が判断）

---

## 登録番号の検証

- 形式: `T` + 13桁の数字（法人の場合は法人番号と同一）
- 国税庁の適格請求書発行事業者公表サイトで確認可能
- URL: https://www.invoice-kohyo.nta.go.jp/
